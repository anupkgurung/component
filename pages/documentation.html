Avatar

Avatar are typically used to display a user profile as a picture by adding class .avatar class Multiple avatars cn used depending on the size by just adding class as .avatar-xsm for extra small or .avatar-lg for large.

Badge

Badges are used to add additional information to any content. Use the .badge class together with a contextual class. Like .badge-status with .avatar

Badges can also added with icons to show notifications on email, outbox etc. .icon-badge class can used to add badge on icon or .text-badge class can added for text based badge.

Image

Images in Bootstrap are made responsive with .responsive. max-width: 100%; and height: auto; are applied to the image so that it scales with the parent element. Responsive image with fix width can also be used by adding .responsive-fix-width class.

The input HTML element is used to create interactive controls for web-based forms in order to accept data from the user; a wide variety of types of input data and control widgets are available.
